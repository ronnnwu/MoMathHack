 Search for the Prime Numbers

This algorithm repeatedly iterates through the numbers 0 to 127, checking each number against a counter, and removing any numbers that are multiples of this counter. The counter begins at 2 and goes up until the square root of 127. For efficiency, the counter is a previously found prime number, as any composite number will by default have already been removed. Example: The initial counter is 2, so any number that is a multiple of 2 (except 2) is removed from the list. So it is with 3, 5, 7, etc.

 Display Binary Numbers

This algorithm displays binary numbers. It does this by treating each of the 5 slat blocks (a block is 20 slats) as value 0 (moved back) or 1 (moved forward). Initially, all the slat blocks are moved back. Then, from right to left, the values of the slat blocks are modified. The first slat block represents 2^0, the second 2^1…the fifth 2^5, so the integers 0 through 32 are represented.

 Demonstrate Bubble Sort

This algorithm repeatedly iterates through each metal slat, the bottoms of which are set completely forward (value of 1), the tops of which are randomly assigned a position from 0 to 1. At each index, the program compares the current ‘top value’ with the next one, wiggling both slats to indicate a comparison, and switching their values if the current is larger than the next. It keeps repeating this process until the slats are organized in increasing order of position.
